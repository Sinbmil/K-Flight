# K-Flight
# 1) 프로젝트 개요
* 게임이름: K-Flight(Knight Flight)
* 장르: 유니티 2D 슈팅게임
* 목표: 적들을 무너트리고 장애물을 피해서 최종보스를 처치하기

# 2) 화면 설명
## 1) 시작화면
* 화면에는 START 버튼과 EXIT 버튼이 있다.
* START 버튼을 누르면 게임이 시작된다.
* EXIT 버튼을 누르면 게임이 종료된다.
* 웅장한 BGM이 나오도록 하였고 무한 루프로 반복되게 설정했다.
